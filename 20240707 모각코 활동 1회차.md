CNN 모델
- 2차원 데이터 (이미지 등)의 패턴을 인식하고 분석하는 데 사용되는 딥러닝 모델.
- 여러개의 층으로 구성됨.

RESNET 모델
- CNN 모델에 잔차 연결 개념을 도입한 것.
- 잔차 연결 : 각 층의 출력을 다음 층으로 직접 보내는 대신에, 입력을 더한 뒤 다음 층으로 전달하는 연결.
